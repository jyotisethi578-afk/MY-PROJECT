Use of Attributes in HTML Tags

Attributes provide additional information about HTML elements. They are always written in the opening tag and usually come in name/value pairs like name="value".

Example 1: href Attribute

The href attribute is used in anchor (<a>) tags to specify the URL of the page the link goes to:
